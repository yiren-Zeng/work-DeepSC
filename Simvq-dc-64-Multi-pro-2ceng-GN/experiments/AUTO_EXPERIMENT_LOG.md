# GN Automatic Experiment Log

## Purpose

- Project copy: `Simvq-dc-64-Multi-pro-2ceng-GN`, copied without saved checkpoint or weight files.
- Required model change: every `BatchNorm2d` in the two-layer U-Net semantic encoder and semantic decoder is replaced with `GroupNorm(num_groups=32, ...)`.
- Evaluation metric: test every saved epoch candidate and the validation-selected best checkpoint on Kodak with no channel; report PSNR and MS-SSIM for the best tested checkpoint.
- Automatic decision target: no-channel `PSNR >= 32.30 dB` and `MS-SSIM >= 0.9777`, matching the target already documented in the source experiment.
- For early manual screening, `SIMVQ_STOP_AFTER_EPOCH=N` pauses an invocation after epoch `N` without shortening the configured 200-epoch learning schedule; the same scheme can then resume its checkpoint.
